# Repo-Local Skills

This repository keeps local AI-agent skills under:

`skills/<skill-name>/SKILL.md`

## Convention

- Use the top-level `skills/` directory for repo-specific skills.
- Create one folder per skill.
- Name each folder with lowercase letters, digits, and hyphens.
- Put the main skill instructions in `SKILL.md`.
- Keep skills scoped to this repository's workflows, content model, or tooling.

## Current skills

- `basdoganlab-website-updater`

## Why this convention

- It keeps repo-specific guidance inside the repository.
- It makes the skill easy to version, review, and share with other collaborators.
- It avoids mixing project-specific instructions into any one person's global agent setup.
